## Formspree Integration

The contact form uses Formspree for form submission. To enable:
1. Update the form action in contact.html:
   ```html
   <form method="POST" action="https://formspree.io/f/YOUR_FORM_ID">
   ```
2. Replace `YOUR_FORM_ID` with your actual Formspree form ID
3. Alternatively, use a different form backend (Netlify Forms, Firebase, etc.)
