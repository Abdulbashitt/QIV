## How It Works

1. User opens gallery
2. script.js loads wallpapers.json
3. Creates URL ‚Üí ID mapping
4. User clicks wallpaper card
5. Gets image URL
6. Looks up wallpaper ID
7. Navigates to /wallpaper.html?id=[ID]
8. wallpaper.html loads
9. Fetches wallpapers.json
10. Finds matching wallpaper
11. Displays full details
12. User can download or share
